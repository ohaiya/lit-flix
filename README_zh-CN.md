## 介绍

LitFlix 是一个书影档案展示及管理平台，用于创建、展示和管理私人书籍、影视档案，支持对档案评分以及添加笔记。

## 技术栈

- React
- TDesign
- Vite

## 开始使用

### 环境要求

- Node.js (v18 或更高版本)
- npm 或 yarn

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/ohaiya/lit-flix.git
```

2. 安装依赖
```bash
cd lit-flix
npm install
# 或者
yarn install
```

3. 启动开发服务器
```bash
npm start
# 或者
yarn dev
```
